Primeiros passos com HTML

Minhas primeiras impressões com o HTML está sendo bem positiva, tendo em vista que estou conseguindo aprender tranquilamente oque foi passado ate agora, entendendo como funciona as tags e tudo que pode ser criado, pois tenho muita vontade de trabalhar com Front End futuramente, já que sempre tive bastante aptidão com a área de design.

Entendendo as Tags

Falando mais sobre as tags, preciso praticar mais para ir pegando os "macetes" e aprofundar mais no universo de possibilidades que o HTML pode oferecer para criação de páginas web, acho bastante interessante todos elementos que pode se criar para a experiencia do usuário.

Explicando comandos

  • h1 ao h6: Servem para criar titulos e subtitulos.
  • p: Servem para criar paragráfos.
  • a: Serve para criar links clicaveis, podendo ser abertos em outras abas ou na própria página principal.
  • li: Serve para criar listas como essa.
  • ul: São usados para listas não ordenadas, sem ordem predefinida.
  • ol: São utilizads para listas ordenadas, ou seja, os itens são apresantados enumerados.
